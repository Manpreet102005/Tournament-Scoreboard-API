# Tournament-Scoreboard-API

A REST API built with Spring Boot and MySQL to manage tournament data.

## Tech Stack
- Java, Spring Boot, Spring Data JPA, MySQL

## Entities
- Player, Team, Match
- Proper cardinality defined between entities

## Features
- CRUD endpoints for all entities
- Input validation
- Global exception handling via @ControllerAdvice
- Structured error responses using ResponseEntity

## Status
Work in progress — score tracking and leaderboard coming next.
